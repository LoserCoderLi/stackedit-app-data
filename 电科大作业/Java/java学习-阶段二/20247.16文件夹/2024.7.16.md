## 自定义异常
![输入图片说明](/imgs/2024-07-16/h4Y5IYdAy4FJLji2.png)

### 步骤
![输入图片说明](/imgs/2024-07-16/bezBq9HjYGQNrLiy.png)

```
public class CustomException_ {  
    public static void main(String[] args) {  
        int age = 180;  
        if (!(age >= 18 && age<=120)){  
            throw new AgeException("年龄需要在18到120之间");  
        }  
        System.out.println("你的年龄在范围之内。");  
    }  
}  
  
// 自定义一个异常  
//1. 一般情况下，我们自定义异常是继承 RuntimeException//2. 也就是把自定义异常做成 运行时异常  
//3. 好处是：我们可以使用默认的处理机制 比较方便  
class AgeException extends RuntimeException{  
    public AgeException(String message) {//构造器  
        super(message);  
    }  
}
```
